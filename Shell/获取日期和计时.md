# 获取日期与计时

## 获取日期

获取时间戳：

```shell
$ date +%s # 使用+指定获取的时间格式
```

时间格式：

| 格式  | 样式                   |
| ----- | ---------------------- |
| %a    | 星期缩写，如Sat        |
| %A    | 星期不缩写，如Saturday |
| %b    | 月份缩写，如Nov        |
| %B    | 月份不缩写，如November |
| %d    | 天，如31               |
| %D    | mm/dd/yy，如10/18/10   |
| %y    | 年份缩写，如10         |
| %Y    | 完整年份，如2010       |
| %I/%H | 小时，如08             |
| %M    | 分钟，如33             |
| %S    | 秒，如10               |
| %N    | 纳秒                   |
| %s    | 时间错                 |

## 时间转化

通过`-d`或者`—date`参数，可以将指定日期，相对的和绝对的都可以。
