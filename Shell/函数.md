# 函数

## 函数定义

函数定义方式：

```shell
function fname()
{
	statements;    
}

# or

fname()
{
    statements;
}
```

## 函数传参

```shell
$ fname arg1 arg2	
```

传入脚本的参数可以通过如下方式访问:

- `$0`是脚本名称
- `$1`是第一个参数
- `$2`是第二个参数
- `$n`是第n个参数
- `$@`和`$*`获取所有参数

虽然`$@`和`$*`都是获取所有参数，但是他们之间有些差别：`"$@"`将参数拓展成`"$1" "$2" "$3" `,`"$*"`将参数拓展成一个字符串`"$1 $2 $3"`

```shell
function fname(){
    for x in "$@"
    do
	echo $x
    done
}

fname ab bc cd

==>
ab
bc
cd

# but

function fname(){
    for x in "$*"
    do
	echo $x
    done
}

fname ab bc cd

==> 

ab bc cd
```
